Cette fonction doit prendre une chaîne de date au format dd/mm/yyyy et la convertir au format yyyy-mm-dd. Si l'entrée n'est pas au bon format, la fonction doit renvoyer un message d'erreur "Error: Invalid date.".

Comment l'utiliser:

  • dans les bases de données, lors de la migration de données d'un système à un autre avec des exigences différentes en matière de format de date;
  • dans les composants de l'interface utilisateur du sélecteur de date, lorsque les données saisies par l'utilisateur peuvent être dans un format différent;
  • dans les outils de reporting pour normaliser les formats de date dans différentes sources de données.

Conditions préalables:

  • l'entrée doit être une chaîne: date ∈ chaîne;
  • l'entrée doit correspondre au modèle: dd/mm/yyyy, où 01 ≤ dd ≤ 31, 01 ≤ mm ≤ 12, et 1900 ≤ yyyy ≤ 2100.
