该函数接收格式为dd/mm/yyyy 的日期字符串，并将其转换为格式为yyyy-mm-dd 的日期字符串。如果输入的格式不正确，函数应返回错误信息"Error: Invalid date." 。

输入：字符串 (str{% if is_js %}ing{% endif %}).

输出： 字符串字符串 (str{% if is_js %}ing{% endif %}).

如何使用：

  • 在数据库中，将数据从一个系统迁移到另一个具有不同日期格式要求的系统；
  • 在日期选择器 UI 组件中，用户输入的可能是不同的格式；
  • 在报表工具中，对不同数据源的日期格式进行标准化。

前提条件

  • 输入应为字符串：日期 ∈ 字符串；
  • 输入应符合以下模式：dd/mm/yyyyy，其中 01 ≤ dd ≤ 31，01 ≤ mm ≤ 12，1900 ≤ yyyy ≤ 2100。
