Diese Funktion sollte einen Datumsstring im Format dd/mm/yyyy entgegennehmen und in das Format yyyy-mm-dd konvertieren. Wenn die Eingabe nicht im richtigen Format ist, sollte die Funktion eine Fehlermeldung "Error: Invalid date." zurückgeben.

Eingabe: Zeichenkette (str{% if is_js %}ing{% endif %}).

Ausgabe: String (str{% if is_js %}ing{% endif %}).

Beispiele: 

{{init_code_tmpl}}

Wie es verwendet wird:

  • in Datenbanken, bei der Migration von Daten von einem System in ein anderes mit anderen Anforderungen an das Datumsformat;
  • in Komponenten der Benutzeroberfläche für die Datumsauswahl, wenn die Benutzereingabe in einem anderen Format erfolgen kann;
  • in Berichtstools, um Datumsformate für verschiedene Datenquellen zu standardisieren.

Vorbedingungen:

  • die Eingabe sollte ein String sein: date ∈ string;
  • die Eingabe sollte dem Muster: tt/mm/jjjj entsprechen, wobei 01 ≤ tt ≤ 31, 01 ≤ mm ≤ 12 und 1900 ≤ jjjj ≤ 2100.
