Funkcja ta powinna pobierać ciąg daty w formacie dd/mm/yyyy i konwertować go do formatu yyyy-mm-dd. Jeśli dane wejściowe nie są w poprawnym formacie, funkcja powinna zwrócić komunikat o błędzie "Error: Invalid date.".

Jak jest używany:

  • w bazach danych, podczas migracji danych z jednego systemu do innego z różnymi wymaganiami dotyczącymi formatu daty;
  • w komponentach UI selektora daty, gdzie dane wejściowe użytkownika mogą być w innym formacie;
  • w narzędziach do raportowania w celu standaryzacji formatów dat w różnych źródłach danych.

Warunki wstępne:

  • dane wejściowe powinny być ciągiem znaków: date ∈ string.
